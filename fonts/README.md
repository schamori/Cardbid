# Fonts Directory

## Genaminto Font

This project uses the **Genaminto** font for headers and display text.

### How to Install Genaminto

1. Download the Genaminto font from one of these sources:
   - https://fontesk.com/genaminto-font/
   - https://fontlot.com/genaminto-font/
   - https://fontmeme.com/fonts/genaminto-font/

2. After downloading, extract the font files and place them in this `fonts` directory with these names:
   - `Genaminto-Regular.ttf` (or .woff, .woff2)
   - `Genaminto-Bold.ttf` (if available)

3. The CSS is already configured to use these font files via @font-face rules in `style.css`
